1. number - define integer data type 
2. String - Stores the String literals we can use ' " `
3. Boolean - Stores the true or false values
4. Undefined - Stores some values until the real value is assigned
5. Null - stores null values

Scopes in js:
1. global scope - scope has globally in the js file, declared outside function or block.
2. function scope - scope has over the function in which it is declared.
3. block scope - scope over the block in which it is declared.
